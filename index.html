Käytetyt kaavat:

R1 = ((R2 * VCC) / V1) - R2

R2 = R1 / ((VCC / V1) -1)

V1 = (R2 * VCC) / (R1 + R2) HUOM! R1 + R2 != 0 ja R2 * VCC != 0

VCC = ((V1 * R1) + (V1 * R2))/R2 HUOM! R2 != 0 ja VCC != 0
